Java Effect Progress Bar:
=======================
Author: Daneel Reventelov 

Description:
---------------------
A graphical progress bar for Java with many effects using gradients in many and particular ways (in future also images), customizable with the help of a test area (execute jar file) to find options to satisfy your taste.
You can also extend a particular class to create your own gradient (instruction in near future). 
This is an Alpha with many bug and few instructions then be carefull!

Italiano (solo descrizione):
Una barra di avanzamento grafica per Java con molti effetti utilizzando i gradienti in molti modi anche non convenzionali (in futuro utilizzerà anche le immagini), personalizzabile con l'aiuto di una area di test (eseguire il file jar) per trovare le opzioni che soddisfano il vostro gusto.
È inoltre possibile estendere una classe particolare per creare il proprio e personale gradiente (istruzione nel prossimo futuro).
Questa è una versione Alpha con molti bug e poche istruzioni per l'uso perciò prestate attenzione nell'utilizzo

Attention:
This is not a branch of IllusionProgressBar project but is totally rewrited with a new idea and a new more powerfull e lightweight engine.
